Contented HTML Text Editor v

Type text and paste items from other applications.

Keys:

  • Ctrl-s (lowercase letter) to save as text, stripping all non-text content.
  • Ctrl-S (capital letter) to save as HTML, preserving all content.
  • Ctrl-o to open files, with automatic detection of text vs. Contented HTML format.

Features:

  • Files saved "as HTML" can be opened in a web browser or reloaded in Contented.
  • Most items pasted into the editor retain their original formatting.
  • Contents are preserved on F5 page refresh.
  • Keyboard navigation.
  • Lightweight and fast, no reliance on external libraries.
Opening a file saved "as HTML" by Contented will restore the entire contents of the editor. Opening a normal ".html" file (or any other file) allows you to edit that file as a text file. Contented looks at the first line of the file to make this determination.

When saving, files are written immediately or a dialog is displayed allowing the name to be changed. This behavior is controlled by the browser. Chrome setting: "Ask where to save each file before downloading". FireFox setting: "Always ask you where to save files". Be contented.
